##  Project Overview

This project involves a comprehensive **Credit Card Fraud Risk Analysis** using transaction data. The goal is to identify fraudulent patterns, assess risk levels, and develop data-driven strategies for prevention. This repository contains the raw data, the detailed analysis report, and an interactive Power BI dashboard for visualization.

---

##  Project Structure

| File Name | Description |
| :--- | :--- |
| `Raw_Card_Data.csv` | The **core dataset** containing credit card transaction details used for all analysis and modeling. |
| `Credit_Card_Fraud_Risk_ Analysis_Dashboard.pbix` | The **interactive Power BI Dashboard** for visualizing fraud trends, risk metrics, and key performance indicators. |
| `Insights_Report.md` | A **detailed report** in Markdown format summarizing the key findings, risk assessment, and strategic recommendations. |

---

##  Key Steps & Methodology

The project followed a structured data analysis pipeline, focused on actionable insights:

| Status | Step | Description |
| :--- | :--- | :--- |
| ✅ | **Data Collection & Cleaning** | Preparing transaction data for fraud analysis, including handling missing values, standardizing formats, and feature engineering. |
| ✅ | **Exploratory Data Analysis (EDA)** | Identifying fraud trends, anomalies, and critical features (e.g., location, time, amount, category) that differentiate fraudulent from legitimate transactions. |
| ✅ | **Building Interactive Power BI Dashboards** | Creating dynamic visualizations for real-time monitoring and in-depth exploration of fraud detection metrics. |
| ✅ | **Risk Assessment Metrics** | Analyzing high-risk transactions and developing metrics to quantify fraud probability and impact. |
| ✅ | **Predictive Insights & Fraud Prevention Strategies** | Deriving data-driven recommendations and strategies for proactive fraud prevention and better decision-making. |

---

##  Technologies & Tools Used

* **Dashboarding & Visualization:** **Power BI**
* **Data Analysis & Reporting:** Markdown, Python (typically Pandas/NumPy for analysis before visualization)

##  How to Use / Navigate This Repository

1.  **View the Data:** The transaction data is available in `Raw_Card_Data.csv`.
2.  **Explore the Insights:** Read the full analysis, findings, and strategic recommendations in the `Insights_Report.md` file.
3.  **Interact with the Dashboard:** Download the `Credit_Card_Fraud_Risk_ Analysis_Dashboard.pbix` file and open it using Power BI Desktop to interact with the full suite of visualizations.
